The Internet of Things (IoT) transforms computing and shapes future smart cities. Despite addressing healthcare system limitations, IoT networks are susceptible to cyber-attacks. This project introduces an ensemble learning-based Intrusion Detection System (IDS) in a fog-cloud architecture for dynamic IoT networks. Leveraging fog computing, it enhances security with granular traffic control and advanced attack detection.

Key Features -

  • Engineered a Resilient Cyber Attack Detection System Empowered by Ensemble Learning
  • Orchestrated an Integrative Approach, Fusing Multiple ML Models for Unparalleled Accuracy
  • Propelled the Meta Classifier, AdaBoost Classifier, into an Ensembled Model, Amplifying its Strength
