The project addresses internet data security by implementing an innovative Text-To-Image Encryption (TTIE) using RGB color values and Diffie Hellman key exchange. Geolocation protection prevents unauthorized access, while a web extension ensures URL safety. Identity and Access Management facilitates group creation for secure chatting. All message components are encrypted, providing a comprehensive solution

Key Features -

  • Text-To-Image Encryption (TTIE) with RGB and Diffie Hellman key exchange
  • Geolocation protection to prevent unauthorized access.
  • Web extension for safe URL detection.
  • Identity and Access Management for secure group creation.
