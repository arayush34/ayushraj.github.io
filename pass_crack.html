The project aims to create an Advanced Password Cracking Tool supporting Bruteforce, Dictionary, and Random Attacks. It targets the cracking of diverse files and hashes, including PDFs, Zip files, MS Office documents, and multiple hash algorithms. Theoretical aspects cover brute force, dictionary, and random attacks, with a practical approach involving the creation of password-protected files. Key features encompass Brute Force, Dictionary, and Random attacks, offering a comprehensive solution for password cracking.

Key Features -

  • Support for Bruteforce, Dictionary, and Random Attacks.
  • Capability to crack password-protected files, including PDFs, Zip files, MS Office documents, and various hashes.
  • User-friendly interface for choosing the attack type and initiating the password cracking process.
  • Practical approach involving the creation of password-protected files and seamless execution of the tool
